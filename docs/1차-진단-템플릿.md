| 3 | #韩国咖啡店 (한국카페) | 범용 키워드 |
| 4 | #济州打卡 (제주핫플) | 인스타형 콘텐츠에 적합 |
| 5 | (업종에 따라 추가) | |

> **확인 방법:** 샤오홍슈 앱에서 해당 키워드를 검색하면 게시물 수와 최근 트렌드를 확인할 수 있습니다.

### 3. 추천 시작 방식

| 추천 | 이유 |
|------|------|
| **파일럿형** (작은 테스트) | 처음이시라면 소규모로 반응을 먼저 확인하는 것을 권장합니다. |

또는 상황에 따라:

- 아직 확신이 없다면 → **진단형** (적합도 분석 + 방향 정리)
- 이미 필요성을 알고 있다면 → **파일럿형** (1차 프로젝트)
- 꾸준한 관리가 필요하다면 → **월운영형** (월 단위 관리)

## 작성 가이드 (내부용)

### 진단서 작성 순서

1. **고객 정보 확인**: 폼/카카오톡에서 업체명, 지역, 업종 확인
2. **샤오홍슈 검색 (5분)**: 앱에서 해당 지역+업종 키워드 검색
   - 검색어 예: `济州咖啡`, `首尔美食`, `釜山海云台`
   - 게시물 수, 최근 게시물 날짜, 콘텐츠 유형 확인
3. **적합도 판단**: 게시물이 활발하면 "적합", 거의 없으면 "검토 필요"
4. **키워드 3~5개 정리**: 검색량 많은 순서로 나열
5. **시작 방식 추천**: 대부분 "파일럿형" 추천
6. **카카오톡으로 발송** 또는 PDF 변환 후 이메일 발송
