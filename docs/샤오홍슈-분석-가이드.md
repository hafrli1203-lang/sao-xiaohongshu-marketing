# 샤오홍슈 분석 가이드 (실전용)

> 고객이 문의하면 이 가이드 순서대로 따라하면 됩니다.
> 소요시간: 10~15분

---

## 1단계. 샤오홍슈 접속 방법

### 모바일 (추천)
1. 앱스토어/구글플레이에서 **"小红书"** (샤오홍슈) 검색 → 설치
2. 전화번호로 가입 (한국 번호 +82 가능)
3. 가입 후 검색창에 키워드 입력

### PC 웹 (가입 없이 가능)
1. 브라우저에서 `www.xiaohongshu.com` 접속
2. 상단 검색창에 키워드 입력
3. 로그인 없이도 게시물 확인 가능 (일부 제한)

> **팁:** 모바일 앱이 더 정확한 결과를 보여줍니다. PC는 빠른 확인용으로 쓰세요.

## 4단계. 검색 결과 분석 (5분)

키워드를 검색한 후 아래 3가지를 확인합니다:

### A. 게시물 수 확인
검색 결과 상단에 **게시물 수**가 표시됩니다.

| 게시물 수 | 판단 | 의미 |
|-----------|------|------|
| 1만 이상 | **적합** | 검색량 많음, 중국인이 이미 관심 갖는 카테고리 |
| 1천~1만 | **보통** | 잠재력 있음, 선점 기회 |
| 1천 미만 | **검토 필요** | 수요가 적거나 키워드를 바꿔야 할 수 있음 |

### B. 최근 게시물 날짜 확인
- 최근 1주일 내 게시물이 있으면 → **활발함**
- 최근 1달 내 게시물 → **보통**
- 3개월 이상 전 게시물만 → **비활성**

### C. 경쟁 콘텐츠 유형 확인
상위 게시물 5~10개를 훑어보며 확인:

- **이미지 위주?** → 사진 촬영이 중요
- **영상 위주?** → 숏폼/릴스 형태 필요
- **어떤 내용?** → 메뉴 소개 / 분위기 / 위치 안내 / 할인 정보
- **좋아요/댓글 수?** → 반응도 확인

## 5단계. 적합도 판단

위 결과를 종합해서 3단계로 판단합니다:

### 적합 ✓
- 게시물 1만 이상
- 최근 게시물 활발
- 비슷한 업종 콘텐츠가 이미 있음
- **→ "파일럿형으로 바로 시작 가능"**

### 보통 △
- 게시물 1천~1만
- 간헐적 게시물
- 비슷한 업종이 일부 있음
- **→ "진단형으로 방향 정리 후 시작 추천"**

### 검토 필요 ✗
- 게시물 1천 미만
- 최근 게시물 거의 없음
- 해당 업종 콘텐츠를 찾기 어려움
- **→ "다른 키워드 접근 필요하거나, 현 시점에서는 효과가 제한적"**

## 6단계. 진단서 작성 (5분)

`docs/1차-진단-템플릿.md`를 열고 아래 내용을 채웁니다:

### 작성 예시 (제주 카페)

```
[OO카페] 샤오홍슈 맞춤 방향 안내

1. 적합도: 적합
   - "济州咖啡" 검색 시 게시물 2만+, 최근 게시물 활발
   - 제주 카페 업종은 중국 관광객의 검색량이 꾸준합니다

2. 예상 키워드
   - #济州咖啡 (제주카페) - 게시물 2만+
   - #济州岛咖啡厅 (제주도카페) - 게시물 8천+
   - #涯月咖啡 (애월카페) - 게시물 3천+
   - #济州打卡 (제주핫플) - 게시물 5만+

3. 추천 시작 방식: 파일럿형
   - 이미지 중심 콘텐츠가 반응이 좋은 편
   - 성수기(6~9월) 전 콘텐츠 확보 추천

4. 참고
   - 상위 게시물은 대부분 매장 분위기 + 시그니처 메뉴 사진
   - 오션뷰/자연 배경이 있는 경우 반응이 특히 좋음
```

## 부록: 시즌별 타이밍 가이드

중국 고객은 연휴/시즌에 맞춰 검색량이 급증합니다.

| 시즌 | 시기 | 검색 급증 시점 | 콘텐츠 준비 시점 |
|------|------|--------------|----------------|
| 춘절 (설날) | 1~2월 | 12월부터 | **11월** |
| 청명절/노동절 | 4~5월 | 3월부터 | **2월** |
| 여름 성수기 | 7~8월 | 5월부터 | **4월** |
| 국경절 (황금연휴) | 10월 첫째 주 | 8월부터 | **7월** |
| 크리스마스/연말 | 12월 | 11월부터 | **10월** |

> **핵심:** 콘텐츠는 **검색 급증 2개월 전**에 준비해야 효과가 있습니다.
> 고객에게 "지금이 OO시즌 준비하기 좋은 타이밍입니다" 라고 안내하면 설득력이 올라갑니다.

## 부록: 업종별 콘텐츠 포인트

고객에게 "어떤 콘텐츠가 효과적인지" 안내할 때 참고하세요.

| 업종 | 잘 되는 콘텐츠 | 핵심 촬영 포인트 |
|------|---------------|----------------|
| 맛집 | 메뉴 클로즈업 + 매장 분위기 | 음식 비주얼, 가격대, 위치 |
| 카페 | 인테리어 + 시그니처 메뉴 | 포토존, 뷰, 특색 있는 메뉴 |
| 뷰티 | 시술 전후 비교 + 과정 | Before/After, 가격, 예약 방법 |
| 숙박 | 객실 투어 + 주변 관광지 | 뷰, 청결, 위치 편의성 |
| 리테일 | 제품 언박싱 + 추천 아이템 | 가격 비교, 한정판, 면세 정보 |
| 체험 | 체험 과정 영상 + 후기 | 재미, 난이도, 소요시간 |

## 요약: 전체 흐름 한눈에

```
고객 문의 접수 (폼/카카오톡)
       ↓
업체명/지역/업종/목표 확인
       ↓
샤오홍슈에서 키워드 3~5개 검색 (이 가이드 3단계 참고)
       ↓
게시물 수 / 최근 활동 / 경쟁 콘텐츠 확인 (4단계)
       ↓
적합도 판단: 적합 / 보통 / 검토 필요 (5단계)
       ↓
진단서 작성 (6단계, 템플릿 사용)
       ↓
카카오톡 또는 이메일로 전달 (7단계)
       ↓
"15분 통화로 자세히 안내드리겠습니다" → 상담 연결
       ↓
상담 후 → 진단형 / 파일럿형 / 월운영형 제안
```
